## Cours 2 : Welcome to the Matrix

### Ma premiere page HTML

```
<!DOCTYPE html>
<html>
<head>
  <title>Le titre </title>
  <meta charset="utf-8"/>
  <link rel="stylesheet" type="text/css" href="main.css">
</head>
<body>
  <h1>Hello World</h1>
</body>
</html>
```
