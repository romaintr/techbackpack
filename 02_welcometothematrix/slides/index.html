Box Model

Tous les éléments HTML peuvent être considéres comme des boites

Les <div> sont des boites qui groupent des élements et structurent le site

Le Box Model permet de structurer la mise en page d'une page HTML

Box Model

Structure d'une <div>

Une boite html est composée des élément suivants:

  • Content: Le texte/image/video à l'intérieur de la balise HTML
  • Padding: L'espace entre le contenu et la bordure
  • Border: La bordure de la boite
  • Margin: L'espace exterieur entre la bordure de la boite et les autres éléments

Box Model

Alignement des boites

Définit le comportement des élements les uns par rapport aux autres.

  • display: block Chaque élement prend une ligne. Ils se rangent les uns en dessous des autres
  • display: inline Les éléments sont affichés sur une meme ligne et apparaissent les uns à coté des autres
  • float: left / right Indique si les éléments doit être placé le plus à gauche ou le plus à droite

Un framework

C'est le framework CSS le plus polulaire qui aide à mettre en forme des pages. Il permet ainsi de concevoir un site web rapidement qui s'affiche sur toute taille d'écran et avec peu de lignes de code ajoutées

Un framework

  • Créer les bases et organise le code
  • Embarque des composants HTML/JAVASCRIPT
  • Organise l'aspect visuel avec un systéme de grille
  • Apporte un style de base aux boutons et formulaires
  • Optimise l'affichage des éléments en fonction des tailles d'écrans

Son histoire

  • Le projet Bootstrap a été créé au départ par Twitter pour répondre à des besoins internes
  • Publié en 2011 il est devenu trés vite populaire
  • Il a été mis à disposition du public sous licence Apache.
  • Le framework en est actuellement à la version 3 dites "mobile first"
  • Il devient critiqué car le design de nos pages web se sont normalisées

Son contenu

  • Une mise en page basée sur une grille de 12 colonnes
  • L'utilisation de Normalize.css pour avoir un rendu identique quel que soit le navigateur utilisé
  • Du code HTML5/CSS3 basé sur le principe du responsive design
  • Des modules javascript de qualité :Carousel, Alert, Modal ...
  • Une documentation
