Créer une page web avec HTML et CSS

Dans cet exercice, nous allons créer une page web simple en utilisant HTML et CSS. Nous allons créer une structure de base pour la page, ajouter du texte, une image et des styles CSS pour personnaliser l'apparence de la page.

Voici les étapes à suivre pour créer la page :

  1. Créez un fichier HTML et nommez-le "index.html".
  2. Dans le fichier HTML, créez une structure de base en ajoutant les balises <html>, <head> et <body>.
  3. Dans la balise <head>, ajoutez une balise <title> et donnez un titre à votre page.
  4. Dans la balise <body>, créez un en-tête <header> et ajoutez-y un titre <h1> avec le nom de votre page.
  5. Ajoutez un paragraphe <p> dans la balise <body> pour décrire brièvement le contenu de votre page.
  6. Ajoutez une section <section> dans la balise <body> pour le contenu principal de votre page.
  7. Dans la section, ajoutez une image en utilisant la balise <img> avec la source de l'image que vous souhaitez afficher.
  8. Ajoutez un sous-titre <h2> et quelques paragraphes <p> pour décrire l'image ou le contenu principal de votre page.
  9. Dans la balise <head>, ajoutez une balise <style> pour ajouter des styles CSS à votre page.
  10. Dans la balise <style>, ajoutez des styles CSS pour personnaliser l'apparence de votre page, par exemple, changez la couleur du texte, la couleur de fond, la police,

L'objectif de cet exercice est de comprendre comment utiliser la balise div pour diviser une page web en sections logiques et comment centrer une div dans une page en utilisant CSS.

Voici les étapes à suivre pour créer la page :

  1. Créez un fichier HTML et nommez-le "index.html".
  2. Dans le fichier HTML, créez une structure de base en ajoutant les balises html, head et body.
  3. Dans la balise head, ajoutez une balise title et donnez un titre à votre page.
  4. Dans la balise body, créez une div avec l'identifiant container.
  5. À l'intérieur de la div container, ajoutez deux autres div avec les identifiants header et content.
  6. Ajoutez du texte dans la div header et du contenu dans la div content.
  7. Dans la balise head, ajoutez une balise style pour ajouter des styles CSS à votre page.
  8. Dans la balise style, ajoutez à la div container la régle suivant pour centrer la div container dans la page.
    width: 80%; margin: 0 auto;
